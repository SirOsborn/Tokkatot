# QR Codes

Quick access QR codes for Tokkatot resources.

## Contents

- **website-qr.png** - Links to [https://tokkatot.aztrolabe.com](https://tokkatot.aztrolabe.com)
- **documentation-qr.png** - Links to project documentation and specifications
- **support-qr.png** - Displays contact information: tokkatot.info@gmail.com
- **demo-qr.png** - Direct access to v1.0 prototype (local network)

## Usage

These QR codes are designed for:
- Print materials and marketing
- Farm equipment labels
- Documentation posters
- Support information displays

### How to Print

All files are high-resolution (300+ DPI) suitable for printing up to A4 size.

### Customization

To regenerate QR codes:
```bash
# Example using qrcode library
pip install qrcode[pil]
python3 generate_qr.py
```
